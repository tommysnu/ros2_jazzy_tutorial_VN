### 1. Các bước tạo ra 1 package
```
mkdir -p ros2_ws/src
cd ros2_ws
colcon build
cd src
ros2 pkg create my_py_pkg --build-type ament_python --dependencies rclpy
```
- Build package 
```
cd ../
colcon build
```
hoặc
```
cd ../
colcon build --packages-select my_py_pkg
```

### 2. Tạo ra 1 node
- Tạo file my_py_pkg/my_py_pkg/my_first_node.py 
```
#!/usr/bin/env python3
import rclpy
from rclpy.node import Node


class MyNode(Node):

    def __init__(self):
        super().__init__("py_test")
        self.counter_ = 0
        self.get_logger().info("Hello world")
        self.create_timer(1.0, self.timer_callback)

    def timer_callback(self):
        self.get_logger().info("Hello " + str(self.counter_))
        self.counter_ += 1

def main(args=None):
    rclpy.init(args=args)
    node = MyNode()
    rclpy.spin(node)
    rclpy.shutdown()

if __name__ == "__main__":
    main()
```
- Kiểm tra lại chính xác hay chưa
```
~/ros2_ws/src/my_py_pkg/my_py_pkg$ python3 my_first_node.py
```

- Vậy làm sao chúng ta có thể thiết lập 1 node 
    - Chúng ta vào phần setup.py
    ```
    entry_points={
            'console_scripts': [
            ],
        },
    ```
    thành
    ```
    entry_points={
        'console_scripts': [
            "py_node = my_py_pkg.my_first_node:main"
        ],
    },
    ```
    - Sau đó build lại package
    ```
    cd ~/ros2_ws
    colcon build --packages-select my_py_pkg
    ```
